vw() function


div {
  width: vw(300px);
  margin-left: vw(30px);
}

When your browser is at 1280px, this div will be 300px wide with a 30px margin left

If you need to have a little more control over the scalable text and want to manually set the lower clamps, use defineType(). In the future I'd love for this to be part of setType() but I'm not sure I'm smart enough to figure it out.


p {
  color: $gray-200;
  @include defineType(14, 16, $minMobileClamp: 14, $minDesktopClamp: 15);
}
