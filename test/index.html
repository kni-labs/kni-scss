vw() function


div {
  width: vw(300px);
  margin-left: vw(30px);
}

When your browser is at 1280px, this div will be 300px wide with a 30px margin left

Sometimes for smaller fonts you want to override the smallest size it can go. In this case pass in the $minClamp argument which is the percentage the minimum font size should be. Set it to 100% to have it not scale any smaller than default.


p {
  @include setType(14, 16, $minClamp: 94%);
}
