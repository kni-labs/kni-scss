vw() function


div {
  width: vw(300px);
  margin-left: vw(30px);
}
When your browser is at 1280px, this div will be 300px wide with a 30px margin left

The new fluid typography


h1 {
  --fontSize: 43;

  @media (min-width: rem($tp)) {
    --fontSize: 56;
  }
}

h1 heading

On desktop, when your browser is set to 1280px, the H1 above will be 56px. It scaleds fluidly until 1600px where it maxes out at 70px. When scaling down from 1280 it will get as small as 47px at 1024.
